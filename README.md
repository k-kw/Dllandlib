# Dllandlib
Cplus_sim，cuda_Cplusで使う動的リンクライブラリを格納しています．
## Overview
「bmp_dll_ver2」ディレクトリにはBMP画像の読み込みや書き込みをするためのクラス・機能がまとまっています．

「complexarray_ver2」ディレクトリには複素数を扱う配列クラスがまとまっており，光学系を想定した数値計算ができます．

「lens_dll_ver2」ディレクトリにはレンズの計算が可能なクラスがまとまっていますが，この機能は「complexarray_ver2」に含まれているので使う必要はありません．

「source」ディレクトリにはライブラリへのコンパイル前のソースコードが格納されています．

「garbage」ディレクトリは使っていません．
